### Premier League AI Analyst

An interactive AI-powered football analytics web app combining machine learning match predictions and an LLM-based insight chatbot. Built with Python & Streamlit, the app allows users to simulate Premier League fixtures and ask football-related questions in real time

## Features 
### Match Prediction Model 
- Predicts the outcome of Premier League matches using:
- Team strength & tendencies
- Venue advantage
- Basic simulated ML logic
- Expected goals & probabilities

### Premier League Insight Bot (LLM)
- An AI chatbot that answers questions about:
- Teams
- Players
- Match history
- Premier League facts & stats
- Uses a Large Language Model (LLM) via secure API integration.

### Modern Streamlit UI
- Interactive dropdowns and buttons
- Clean layout & responsive design
- Smooth deployment on Streamlit Cloud

## Tech Stack
- Python
- Streamlit
- Pandas / NumPy
- Google Gemini / OpenAI API (LLM)
- Streamlit Cloud Deployment
  
## How to run locally
### Clone the repo
git clone https://github.com/Adnan-Ahmad-Khan/Premier_League-AI-Analyst.git
### Enter project folder
cd Premier_League-AI-Analyst
### Install dependencies
pip install -r requirements.txt
### Run the app
streamlit run app.py
